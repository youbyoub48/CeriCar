# CeriCar

CeriCar est une application web de covoiturage développée avec le framework Yii2.

## Fonctionnalités

- Utilisateurs : Gestion de profil.
- Conducteurs : Proposition de trajets et de véhicules.
- Passagers : Recherche et réservation de trajets.

## Base de données et Accès

L'application est configurée pour se connecter à une base de données PostgreSQL hébergée par l'université.

⚠️ Important : L'accès à la base de données nécessite d'être connecté au VPN de l'université.
